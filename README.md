symbolhash
==========

Given a string, returns a unicode hash of the desired length

Example
-------

Example input and output:

"Several different words lalalalala" -> ♥☻☘⇧❤❄תּ☂☢♞✂♻⚡▷☮♥
"hi" -> ☭☢☭☢☭☢☭☢☭☢☭☢☭☢☭☢

The entropy is reflected in how much repetition there is in the resulting hash.
